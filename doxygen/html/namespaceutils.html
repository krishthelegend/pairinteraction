Generate a random integer.

The function generates a random integer in the closed interval [a,b].

Thread-local static random engine.

To save some effort this function initializes a static thread-local random engine to avoid race conditions and broken random sampling. It is seeded once using std::random_device for good entropy.

Returns
Reference to static thread-local random engine
